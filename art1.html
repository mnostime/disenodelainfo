Sitio web 101

Instrucciones para crear una página web

Septiembre 1, 2022 Montse Nosti

Lo primero que hay que hacer es entrar al sitio web de html5up.net y buscar una plantilla que nos acomode y nos agrade. La idea es encontrar una que tenga interacción. Esto quiere decir que cuenta con más de una página. Damos descargar.

Después de descargar el código, abrimos la aplicación de Visual Studio Code. Damos en File, luego en Open Folder y veremos del lado izquierdo las diferentes pestañas con las que cuenta. Si esto no aparece, hay que hacer clic en Explorer.

La página más importante es la que se llama index.html. Es importante no cambiarle el nombre. Vamos a ver un código con distintas secciones. Para hacer modificaciones, lo más sencillo es ir explorando el código y la página al mismo tiempo. Así podremos identificar en qué parte del código se encuentra cada sección y la podremos modificar. Sirve buscar los elementos con CTRL+F. Por ejemplo, si copiamos y pegamos este texto, podemos encontrar la parte del código donde se encuentra de forma más sencilla. Las secciones que no sirvan, hay que eliminarlas.

¿Cómo agregar imágenes?

Para agregar imágenes hay que descargarlas y moverlas a la carpeta de images de nuestro sitio. Después de identificar dónde queremos poner la imagen en el código escribimos img src="images/nombredelaimagen.jpg/png" y enseguida escribimos alt= y escribimos el texto alternativo entre comillas. Esto sirve para que si la imagen no carga, aparezca el texto o para que una persona invidente pueda saber qué contiene la imagen. Es importante revisar que la terminación del archivo sea la correcta.

¿Cómo agregar vínculos?

Para agregar un vínculo hay que colocar la función a href= y el vínculo hacia el que queremos dirigir entre comillas. A continuación, colocamos el texto que queremos que redireccione al vínculo y cerramos la función con /a.
